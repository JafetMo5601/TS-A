This is the home page of TS&A project.


The TS&A project by it's acronym Translator, Speech and Analyzer is a web app was created to have a tool capable to uses Azure for text translation, text-to-speech conversion, and sentiment analysis of input text and translations. Also built this app was a good way to learn more about azure services and other technologies. Learn more about Azure Cognitive Services. Some technologies used in this quick project:

  1. Flask for back-end development.
  2. Azure Cognitive Servces
  3. JavaScript and Ajax.
  4. HTML5/CSS3/Bootstrap4.
  5. JQuery to execute the requests.
